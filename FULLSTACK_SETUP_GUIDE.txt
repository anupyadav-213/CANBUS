PROBLEM 1: "Port 5000 already in use"
Solution:
  Option A: Kill the process using port 5000
    Windows: netstat -ano | findstr :5000
    Mac/Linux: lsof -i :5000
  
  Option B: Run on different port
    In app.py, change: app.run(port=5001)
    Then open: http://localhost:5001

PROBLEM 2: "ModuleNotFoundError: No module named 'flask'"
Solution:
  $ pip install flask
  Or reinstall all: pip install -r requirements_fullstack.txt

PROBLEM 3: "Database is locked"
Solution:
  Delete: can_analyzer.db
  Restart the application
  Database will be recreated

PROBLEM 4: "No module named 'src'"
Solution:
  Make sure src/ folder is in the same directory as app.py
  Or create __init__.py file: src/__init__.py

PROBLEM 5: "404 Page Not Found"
Solution:
  Make sure templates/ folder exists
  Make sure index.html and dashboard.html are in templates/
  Restart the application

PROBLEM 6: "Login doesn't work"
Solution:
  Make sure you registered first
  Check that username and password match exactly (case-sensitive)
  Check browser console for error messages (F12)
